## 起動方法

```bash
# 仮想環境を有効化
source venv/bin/activate  # Mac/Linux
# または
venv\Scripts\activate  # Windows

# GUIを起動
python run_gui.py
主な機能
1. ファイル選択
個別ファイル選択
フォルダ一括選択
ドラッグ&ドロップ（予定）
2. 変換モード
単一ファイル変換: 1つのHTMLを1つのWordに
統合変換: 複数のHTMLを1つのWordに統合
3. HTML検証
リアルタイム検証
エラー・警告表示
構造チェック
4. プレビュー機能
変換前のプレビュー
ログ表示
進捗表示
5. 出力設定
出力先の指定
ファイル名の自動生成
既存ファイルの上書き確認
使用方法
基本的な流れ
GUIを起動

Copypython run_gui.py
入力ファイルを選択

「ファイルを選択」または「フォルダを選択」をクリック
複数ファイルを追加可能
変換モードを選択

単一ファイル変換：最初のファイルのみ変換
統合変換：すべてのファイルを1つの文書に統合
出力先を指定

「選択」ボタンをクリック
.docxファイルとして保存
（オプション）HTMLを検証

「HTMLを検証」ボタンをクリック
エラーや警告をチェック
変換開始

「変換開始」ボタンをクリック
進捗バーで状態を確認
完了後、ファイルを開く
スクリーンショット

トラブルシューティング
問題: GUIが起動しない
原因: PySide6がインストールされていない

解決策:

Copypip install PySide6
問題: ファイルが選択できない
原因: ファイルパスに日本語が含まれている

解決策: 英数字のパスを使用

問題: 変換が遅い
原因: 大量のファイルまたは複雑な数式

解決策: ファイル数を減らすか、DPI設定を下げる
